Tags

  2. b tag - makes things bold
  3. em tag - stands for emphasis
    • Puts items in italic
  4. img tag - for including images in html
    • Source (src) which equals URL - url of image
  5. Alt, alternate - which displays text in event or error in URL imag WhiteSpace
  6. Br equals break - break in line of code
  7. P equals paragraph - ie new paragraph
  8. Inline versus Block
    • Br tag is inline
    • P tag is Block
  9. Span & Div
    • Span is inline
    • Span will run code together as would be anticipated
  10. Div is Block
    • Div will block out a section of code, seperate it from remainder, ie block it out
  11. HTML Documents
    • !doctype html: document type
